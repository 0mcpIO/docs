The playground is a chat screen for trying tools, resources, and prompts from your published version.

## Use the playground

1. Publish a version of your server.
2. Open **Playground** from the server menu.
3. Open settings and add an AI provider key if requested.
4. Add the upstream API credential required by your tools.
5. Ask for a simple task, such as listing one record.
6. Open the playground logs to review the MCP calls.

Start with a read-only tool. Check the input and result before trying an operation that creates, updates, or deletes data.

<Note>The AI provider key stays in browser session storage. 0mcp does not save playground credentials or request payloads in activity records.</Note>
