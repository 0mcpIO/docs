<Info>GraphQL schema import is planned and is not available yet.</Info>

0mcp does not currently turn GraphQL queries and mutations into MCP tools. If your service also provides REST endpoints, describe those endpoints with OpenAPI and import them today.

Do not upload a GraphQL schema in the OpenAPI importer. It will fail validation because the formats are different.
