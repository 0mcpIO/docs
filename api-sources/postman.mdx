<Info>Postman Collection import is planned and is not available yet.</Info>

If your collection can be converted to an OpenAPI document, review the result and import it through [OpenAPI specification](/api-sources/openapi). Check the server URL, authentication, input types, and operation names after conversion.

Do not upload a Postman Collection JSON file directly. The OpenAPI importer expects an OpenAPI or Swagger version field.
