title: Introduction
description: Learn what 0mcp does and when to use it.
---

0mcp turns an existing API into a hosted [Model Context Protocol](https://modelcontextprotocol.io/) server. This lets an MCP client, such as Claude Code or Cursor, find and call your API operations as tools.

You do not need to build or host a separate MCP server. Give 0mcp an OpenAPI 3.x or Swagger 2.0 document, review the operations, and publish the ones you want to share.

## What you get

- A hosted MCP endpoint
- Tools made from selected API operations
- Resources for fixed content
- Prompts with reusable inputs
- Saved versions that you can publish or restore
- Usage numbers and activity logs
- A playground for trying capabilities before client setup

## What you keep

Your upstream API stays in place. The client sends the needed Bearer token or API key when it makes a request. 0mcp passes that value to your API for the call.
